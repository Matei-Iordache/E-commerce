GameGrid

Aplicatie e-commerce B2C pentru jocuri, construita cu Node.js, Express si SQLite.

Aceasta pagina este publicata pe GitHub Pages. Aplicatia completa ruleaza pe server Node.js (nu direct pe Pages).

Functionalitati principale

  • Catalog, categorii, cautare si pagina produs
  • Cos de cumparaturi si checkout
  • Plata prin Stripe Checkout
  • Factura si istoric comenzi
  • Admin pentru produse, comenzi, utilizatori si stoc

Rulare locala

npm install

npm start

Aplicatia devine disponibila la http://localhost:3000.

Deployment recomandat pentru aplicatia completa

Pentru rularea completa (backend + baza de date), foloseste un hosting Node.js: Render, Railway, Fly.io, VPS sau platforma similara.
